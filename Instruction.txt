Test the Camera on Raspberry Pi

    export DISPLAY=:0
    libcamera-hello -t 0


Installing Dependencies and Creating a Folder on Raspberry Pi

    sudo apt update && sudo apt upgrade -y
    mkdir yolo  # Create a folder
    cd yolo
    python3 -m venv --system-site-packages venv  # Create a virtual environment
    source venv/bin/activate  # Activate the virtual environment


Installing YOLOv8 for Raspberry Pi

    cd yolo
    source venv/bin/activate  # Activate the virtual environment
    pip install ultralytics ncnn


Download a Model from COCO that Includes yolov8n.pt

    yolo detect predict model=yolov8n.pt  # This will open the model


Export YOLO to NCNN Format

    yolo export model=yolov8n.pt format=ncnn


Download the Object Detection Code (Credits to ejtech.io)

    wget https://ejtech.io/code/yolo_detect.py


Sample Command to Run It

    python yolo_detect.py --model=yolov8n_ncnn_model --source=picamera0 --resolution=640x480
